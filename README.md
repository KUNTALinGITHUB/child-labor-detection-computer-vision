# An Integrated Multi-Stage Computer Vision System for Real-Time Detection of Potential Child Labor in Hazardous Industrial Environments

**Authors:** Prerana Basak, Kuntal Pal, Arup Dutta  
**Journal:** International Research Journal of Modernization in Engineering Technology and Science (IRJMETS)  
**Volume:** 08, Issue 01, January 2026  
**DOI:** https://doi.org/10.56726/IRJMETS88292  

## Abstract
This paper presents a multi-stage real-time computer vision system for detecting potential child labor in hazardous industrial environments such as construction sites, industrial areas, and workshops. The system integrates scene classification (ResNet50), object and PPE detection (YOLO), face detection, demographic age estimation (FairFace), and pose-based work activity recognition to enable proactive monitoring and real-time alerts.
